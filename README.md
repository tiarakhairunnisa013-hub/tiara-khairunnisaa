# Tiara Khairunnisaa - projek uts
Sistem Manajemen Data Mahasiswa berbasis web dinamis yang dibangun murni menggunakan HTML5, CSS3, dan Vanilla JavaScript (Tanpa Framework) untuk memenuhi tugas UTS Pemrograman Web. Mengimplementasikan fitur CRUD, Pencarian, Filter Gender, Paginasi Dinamis, Mode Gelap, dan penyimpanan persisten berbasis Local Storage.
dilakukan dengan berkelompok : 1. Mutiara Ramadhani (ketua), 2. Tiara Khairunnisa, 3. Nurhidayah.
